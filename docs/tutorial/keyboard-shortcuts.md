# Keyboard Shortcuts

Master these shortcuts to work faster in OpenAlgo Flow.

---

## Editor Shortcuts

### Node Operations

| Shortcut | Action |
|----------|--------|
| `Delete` or `Backspace` | Delete selected node or edge |
| `Escape` | Deselect current selection |
| `Click` on node | Select node |
| `Click` on edge | Select edge |

### Workflow Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl + S` | Save workflow |
| `Ctrl + Z` | Undo (coming soon) |
| `Ctrl + Y` | Redo (coming soon) |

### Navigation

| Shortcut | Action |
|----------|--------|
| `?` | Open keyboard shortcuts page |
| `Scroll` | Zoom in/out |
| `Click + Drag` on canvas | Pan view |

## Connection Editing

### Creating Connections

1. Hover over a node handle (small circle)
2. Click and drag to another node's handle
3. Release to create connection

### Deleting Connections

1. Click on the edge (line) to select it
2. Press `Delete` or `Backspace`

---

## Selection

### Single Selection

- Click on a node to select it
- Click on an edge to select it
- Click on empty canvas to deselect

### Multi-Selection (Coming Soon)

- `Shift + Click` to add to selection
- `Ctrl + A` to select all

## Quick Actions

### Save Workflow

```
Ctrl + S
```

Saves the current workflow. The save button in the header also works.

### Delete Selected

```
Delete  or  Backspace
```

Deletes the currently selected node or edge.

### Deselect

```
Escape
```

Clears the current selection.

### Open Shortcuts Help

```
?  or  Shift + /
```

Opens this shortcuts reference page.

## Tips for Efficiency

1. **Use Tab Key**: Navigate between input fields in the Properties panel

2. **Right-Click Menu**: Access context menu on nodes (coming soon)

3. **Drag Multiple**: Hold Shift while dragging to duplicate (coming soon)

4. **Quick Save**: Get in the habit of `Ctrl+S` after changes

5. **Keyboard Navigation**: Use Tab and Enter in forms
